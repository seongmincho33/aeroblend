## 그룹 요약

두 연구소의 공동 연구를 통해 마우스 에임 시스템의 수학적 기반을 **완전히** 분석하고, **P0~P3 우선순위의 14개 개선 항목**을 도출했다. 최종 산출물은 **1634줄, 20개 섹션**의 종합 수학 공식집이다.

### 3대 핵심 발견

1. **V³ 루프 게인 불일치**: gain_scale ∝ V¹이나 공기역학적 토크 ∝ V² → 루프 게인이 V³으로 발산. 해결: `gain_scale = clamp((Vref/V)², 0.3, 3.0)`
2. **D_scale 공식 오류**: `sqrt(ca/ca_ref)` → `(ca_ref/ca)^(3/4)` (방향 반전). 복엽기 ζ=0.18 원인 규명
3. **control_authority 평균관성 오류**: 축별 관성 모멘트를 평균하여 F-16에서 4.2배 오차 발생

### 최종 산출물

`docs/mouse-aim-math-formulas.md` — 20개 장 + 2개 부록의 종합 수학 공식집

## 회사별 요약

### alpha - 제어이론 연구소

**연구 범위**: PD 제어기의 전달함수 모델, 2차 시스템 분석, 안정성 증명, 최적 게인 도출

**주요 성과**:
1. **PD 제어기 전달함수 유도**: 플랜트 P(s) = K_c / (s(s+b)), 폐루프 2차 시스템 도출
2. **현재 감쇠비 분석**: ζ=0.47 (최적 0.7 대비 심각하게 부족, 오버슈트 19.5%)
3. **최적 Kd 공식**: Kd* 유도, α=2.0 최종 확정, gain_scale ≤ 3.0 클램핑
4. **안정성 증명**: 위상여유(PM) = 62°, D_scale 수정 후 ζ=0.602 달성
5. **D항 필터 설계**: τ = 0.008~0.016s, 고주파 잡음 감소, 위상여유 영향 최소
6. **3-Body Convergence 수학**: 마우스→카메라→항공기 3체 수렴의 수학적 모델 확립
7. **D_scale 오류 발견**: `sqrt(ca/ca_ref)` → `(ca_ref/ca)^(3/4)` 수학적 증명
8. **ca 축별 분리**: 평균관성 대신 축별 관성으로 F-16 4.2배 오차 해결
9. **Reference Windup**: back-calculation anti-windup 공식 도출

### beta - 항공역학 연구소

**연구 범위**: 협조선회 수학, 실속 보호, 에너지 관리, 항력 극선, 비행감 향상

**주요 성과**:
1. **협조선회 수학 모델**: 이상적 선회 공식, K_turn(V) = g·tan(φ_desired)/V 유도
2. **속도 의존 선회 이득**: bank_gain(V) 속도 의존 공식 (고정 1.5 대신)
3. **에너지 관리 모델**: 비에너지 E_s = h + V²/(2g), Ps 프로파일별 수치 테이블 (An-2/C172/F-16)
4. **실속 보호 수학**: V_stall_turn = V_stall × √n, 하중계수 보정 포함
5. **항력 극선**: CD = CD0 + CL²/(πeAR), 오스왈드 효율 계수 보정
6. **P0 버그 발견**: 60° 선회 시 오토스로틀이 실속 방지 실패 → n-adjusted 공식 도출
7. **gain_scale 확정**: clamp((Vref/V)², 0.3, 3.0) — 양팀 최종 합의
8. **사이드슬립 보정**: PD 사이드슬립 댐퍼 rudder = -0.8β - 0.3β̇
9. **비행감 향상**: G-force 계산 수정, 카메라 하중계수 변조 공식
10. **6개 누락 피드백 루프 식별**: F1-F6 피드백 경로 정의
11. **지속선회 한계**: 54° 뱅크각 물리적 한계 수학적 유도

## 예상 효과

### P0: 즉시 수정 필수
- **n-adjusted 오토스로틀**: 60° 선회 시 실속 방지 실패 해결
- **gain_scale = clamp((Vref/V)², 0.3, 3.0)**: V³ 루프 게인 발산 방지

### P1: 즉시 적용 권장
- **D_scale 수정**: `(ca_ref/ca)^(3/4)` → 모든 기체 ζ ≈ 0.7 일관 유지
- **ca 축별 분리**: F-16 등 비대칭 관성 기체의 4.2배 오차 해결
- **Kd 25% 증가**: 오버슈트 19.5% → ~4.6%, 정착시간 단축

### P2: 중기 개선
- **Reference Windup 방지**: back-calculation anti-windup
- **사이드슬립 댐퍼**: 선회 중 방향 안정성 향상
- **D항 필터 (τ=0.01s)**: 조종면 미세 진동 감소

### P3: 장기 개선
- **속도 의존 bank_gain**: 고속 급선회, 저속 안정 선회
- **G-force 카메라 피드백**: 선회 시 시각적 하중감 표현
- **6개 피드백 루프 통합**: 완전한 비행제어 시스템 구축
